事件 ​

要为元素绑定事件处理程序，只需要在事件类型前面加上@符号，然后将函数名填入即可，如下示例：

vue
<script setup>
        function a(){
            document.body.innerHTML+='hello,world'
        }

</script>

<template>
        <button @click="a">点我</button>
</template>

也可以为处理程序加入参数：

vue
<script setup>
function a(msg){
    document.body.innerHTML += msg
}

</script>

<template>
<button @click="a('hello,world')">点我</button>

</template>
